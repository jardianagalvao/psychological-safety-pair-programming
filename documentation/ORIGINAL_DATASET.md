# Original Dataset

This study reuses data from the PP-ind repository, a public research dataset
containing audiovisual recordings, transcripts, contextual annotations, and
metadata from industrial pair programming sessions.

## Access to the original dataset

The original PP-ind dataset is available on Zenodo:

https://zenodo.org/records/4541509

The audiovisual recordings, complete transcripts, and original metadata are
not redistributed in this replication package. Researchers interested in
accessing the primary data should consult the original repository.

## Dataset used in this study

The study analyzed the publicly available transcripts from the following
pseudonymized industrial contexts:

- Company A
- Company B
- Company C
- Company D
- Company E
- Company J
- Company O
- Company P

Only sessions with publicly available transcripts were included.
